  1. If the sample size is large enough (n > 30), we can ignore the distribution of the data and use parametric tests.

  2. The central limit theorem tells us that no matter what distribution things have, the sampling distribution tends to be normal if the sample is large enough (n > 30).

  3. However, to be consistent, normality can be checked by visual inspection [normal plots (histogram), Q-Q plot (quantile-quantile plot)] or by significance tests.

  4. Density plot and Q-Q plot can be used to check normality visually.

require(ggpubr)

  1. Q-Q plot (or quantile-quantile plot) draws the correlation between a given sample and the normal distribution. A 45-degree reference line is also plotted.
require(ggpubr)
ggqqplot(my_data$len, color = 'red')

stats::qqnorm(my_data$len)

  1. There are several methods for normality test such as Kolmogorov-Smirnov (K-S) normality test and Shapiro-Wilk’s test.

  2. The null hypothesis of these tests is that “sample distribution is normal”. If the test is significant, the distribution is non-normal.

  3. Shapiro-Wilk’s method is widely recommended for normality test and it provides better power than K-S.

  4. It is based on the correlation between the data and the corresponding normal scores.

  5. Note that, normality test is sensitive to sample size. Small samples most often pass normality tests. Therefore, it’s important to combine visual inspection and significant test in order to take the right decision.

  6. The R function shapiro.test() can be used to perform the Shapiro-Wilk test of normality for one variable (univariate):

stats::shapiro.test(my_data$len)
